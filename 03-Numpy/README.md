# 🧮 03 - NumPy: Scientific Computing & Matrix Operations

Welcome to the NumPy module of the AI/ML Learning journey! This folder contains my foundational work with **NumPy (Numerical Python)**, the core library for scientific computing in Python. 

Before diving into complex machine learning pipelines and pandas dataframes, mastering NumPy is essential for understanding how data is structured, manipulated, and processed mathematically under the hood.

### [My_Numpy_Notebook.ipynb](https://github.com/vikassinngh123/AI-ML-Learning/blob/main/03-Numpy/My_Numpy_Notebook.ipynb)
This comprehensive Jupyter Notebook serves as both a tutorial and a reference guide for manipulating multidimensional array objects. 

**Key Concepts Mastered:**
* **N-Dimensional Arrays (`ndarray`):** Creating and inspecting 1D vectors, 2D matrices, and 3D tensors.
* **Array Indexing & Slicing:** Extracting specific data subsets, filtering with boolean masks, and conditional selections.
* **Broadcasting Mechanics:** Implementing advanced 3x3 array broadcasting to perform operations on arrays of different shapes without memory-heavy data replication.
* **Reshaping & Restructuring:** Using `.reshape()`, `.flatten()`, and array concatenation to format data for machine learning model inputs.
* **Mathematical & Statistical Operations:** Aggregations (min, max, mean, sum), dot products, and element-wise matrix math.

## 🚀 Why NumPy?
In Machine Learning, everything (images, text, audio, CSV data) is eventually converted into numbers. NumPy provides the highly optimized, C-based arrays and mathematical functions necessary to process these massive datasets fractions of a second faster than standard Python lists.
