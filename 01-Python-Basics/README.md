#  01 - Python Basics

Welcome to the **Python Basics** directory! 

This folder serves as the programming foundation for my broader AI and Machine Learning journey. Before diving into complex data structures and algorithms, this section focuses on mastering core Python concepts, control flow, and logic-building through interactive, terminal-based games.

##  Project Showcase

Here are the beginner-friendly mini-projects currently in this folder:

* **[`casino/`](./casino):** A command-line Roulette game complete with core betting options, randomized outcomes, and payout logic.
* **[`guess_the_number.py`](./guess_the_number.py):** An interactive number guessing game featuring 3 distinct play modes to test algorithmic logic.
* **[`rock_paper_scissors.py`](./rock_paper_scissors.py):** The classic game built with continuous score tracking and robust user input validation to prevent crashes.

##  Core Skills Demonstrated

While these are fun games, they were specifically built to solidify fundamental software engineering concepts, including:
* **Control Flow:** Advanced use of `if/elif/else` statements and `while`/`for` loops.
* **State Management:** Tracking scores, player bankrolls, and game states across multiple rounds.
* **Input Validation:** Sanitizing raw user inputs to ensure the programs do not break unexpectedly.
* **Modularity:** Organizing logic into reusable functions.
* **Standard Library Usage:** Utilizing Python's built-in modules (like the `random` library).
